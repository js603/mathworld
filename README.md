# MathWorld 🌍🔢

수학적 개념을 기반으로 한 텍스트 기반 RPG 시뮬레이션 게임입니다.

## 개요

MathWorld는 수학적 모델링을 활용하여 동적인 게임 세계를 시뮬레이션합니다. 캐릭터 관계, 경제 시스템, 생태계, 날씨 등이 수학적 알고리즘에 의해 상호작용하며 변화하는 살아있는 세계를 경험할 수 있습니다.

## 주요 기능

- **🧠 Utility AI**: 효용 함수 기반의 지능적인 NPC 의사결정
- **🔗 관계 그래프**: 캐릭터 간 동적 관계 시스템 (신뢰, 우정, 적대 등)
- **⚔️ 전투 시스템**: 확률 기반 전투 메커니즘
- **🌦️ 날씨 시뮬레이션**: 동적 기상 시스템
- **💰 경제 시스템**: 수요/공급 기반 경제 모델
- **🌿 생태계**: 포식자-피식자 동역학
- **🦠 질병 시뮬레이션**: 전염병 확산 모델
- **📖 내러티브 생성**: 동적 이벤트 및 대화 생성

## 기술 스택

- **Language**: TypeScript
- **Runtime**: Node.js
- **Testing**: Jest

## 프로젝트 구조

```
mathworld/
├── src/
│   ├── core/           # 핵심 시스템 (캐릭터, 관계, AI)
│   ├── game/           # 게임 로직 (전투, 게임 루프)
│   ├── narrative/      # 내러티브 시스템 (이벤트, 텍스트 렌더링)
│   ├── simulation/     # 시뮬레이션 (날씨, 경제, 생태계, 질병)
│   └── utils/          # 유틸리티 (그래프, 수학, 랜덤)
├── tests/              # 테스트 파일
└── data/               # 게임 데이터 (캐릭터, 위치, 템플릿)
```

## 설치

```bash
# 의존성 설치
npm install

# TypeScript 빌드
npm run build
```

## 실행

```bash
# 인터랙티브 CLI 게임 플레이
npm run play

# 데모 실행
npm run demo

# 분석 데모 실행
npm run analyze

# 개발 모드
npm run dev
```

## 테스트

```bash
# 전체 테스트 실행
npm test

# 감시 모드로 테스트
npm run test:watch
```
